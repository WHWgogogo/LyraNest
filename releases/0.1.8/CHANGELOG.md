# LyraNest 0.1.8

发布日期：2026-07-21

## 主要更新

- 完成 LyraNest主要功能
- 修复断网启动后恢复联网仍无法播放和加载歌词的问题。
- 完善离线下载歌曲的封面、歌词和离线播放能力。
- 修复 Android 通知栏播放模式与桌面歌词按钮错位问题。
- 优化 Android 桌面歌词背景宽度，使其跟随歌词内容动态变化。
- 提供 Android ARM64、Windows x64 与 Docker 服务端发行版本。

## Docker 镜像

- 固定版本：`ghcr.io/whwgogogo/lyranest-server:0.1.8`
- 最新版本：`ghcr.io/whwgogogo/lyranest-server:latest`
- 发行展示名称：`LyraNest-server-0.1.8`
